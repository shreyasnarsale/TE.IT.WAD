  * Practical (PR): 25 Marks
  * Term Work (TW): 50 Marks

---

## 🎯 Course Objectives

1. To understand basic concepts of web programming and scripting languages.
2. To learn Version Control Environment.
3. To learn front-end and back-end technologies.
4. To understand mobile web development.
5. To comprehend web application deployment.

---

## 🎓 Course Outcomes

After completion of this course, students will be able to:

* **CO1:** Develop static and dynamic responsive websites using HTML, CSS, Bootstrap, and AJAX.
* **CO2:** Create version control environment using Git and GitHub.
* **CO3:** Develop applications using frontend and backend technologies.
* **CO4:** Develop mobile websites using jQuery Mobile.
* **CO5:** Deploy web applications on cloud platforms such as AWS.

---

## 🧪 List of Laboratory Assignments

### 🔹 Assignment 1

* Create responsive web page (dashboard) using HTML, CSS, Bootstrap
* Implement user registration using JavaScript, AJAX, and local storage

---

### 🔹 Assignment 2

* Create GitHub account and push code using Git commands
* Create Docker container environment
* Develop Angular application with:

  * User Registration
  * User Login
  * Profile Component

---

### 🔹 Assignment 3

* Develop Node.js application to serve static website
* Create CRUD APIs using Node.js, Express.js, and MongoDB

---

### 🔹 Assignment 4

* Develop mobile website using jQuery Mobile
* Deploy web application using AWS

---
